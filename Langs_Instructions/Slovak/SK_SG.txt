# Slovak Style Guide for Skyscanner

## Critical Requirements
**Tone Hierarchy**: **Helpful** > **Playful** > **Geeky**. If a conflict exists, prioritize clarity and helpfulness over tone.
**Helpful**: Make complex info simple; clarify confusing situations; make overwhelming choices manageable.
**Playful**: Use a conversational, natural rhythm; avoid "fake-smart" corporate speak or stiff phrases.
**Geeky**: Show expertise, share data in a human way, and explain patterns simply.
**Inclusive Language**: Avoid stereotypes, gendered assumptions, or labels. Use gender-neutral terms where possible.
**No Brand Translation**: Never translate "Skyscanner." Do not add articles unless grammatically required.

## Technical Rules
### URL Requirements
- **No spaces**: Use hyphens (`-`) to separate words.
- **Lowercase**: Use all lowercase.
- **Character Set**: No extended/special characters. Replace with standard Latin letters.

### Placeholders & Variables
- **Do not translate, delete, or change** placeholders/tags (e.g., `@@city@@`).
- **Preserve order**: Keep exact order as source.
- **Grammatical Concordance**: Ensure sentences remain natural. Use workarounds (e.g., adding descriptors like "mesto" or using a colon) if the placeholder forces incorrect case/grammar.
- **Spacing**: Ensure spaces exist before/after tags to prevent text collision.

## Language-Specific Guidelines
### Punctuation & Spacing
- **Exclamation Marks**: Do not overuse. Avoid "shouting."
- **Commas**: Follow Slovak grammar rules; do not blindly mirror source punctuation.
- **Quotation Marks**: Use standard Slovak double quotation marks („...“).
- **Dashes**: Use EN-dash (–) for ranges/explanations; do not use EM-dash.
- **Full Stops**: Omit at the end of fragments (titles, buttons, UI elements, email subjects). Use for sentences/headings consisting of multiple sentences.

### Capitalization
- **Sentence Case**: Prefer sentence case for titles/CTAs. Avoid title casing (e.g., "Book Now" -> "Rezervujte si").
- **Months/Days**: Use lowercase even as standalone items.
- **Proper Nouns**: Capitalize per standard Slovak rules.
- **Colons**: Use lowercase after a colon for explanatory statements; uppercase for lists/quotes/standalone sentences.

### Formality (T/V Policy)
- **Formal 'Vy'**: Use the formal "you" in Slovak copy.
- **Capitalization**: Capitalize "V" (Vaša/Vám) in direct communications (emails, notifications) to specific users.

### Translation Nuances
- **Anglicisms**: Avoid unless the term is widely used/expected (e.g., well-known digital terms).
- **Abbreviations**: Use only if widely understood. If in doubt, write the full term on first mention with the abbreviation in brackets.
- **Active Voice**: Use active voice, especially in headlines/CTAs. Avoid passive constructions.
- **Transliteration**: Follow official standards for place names. Do not "localize" brand names. Use standard exonyms if established; otherwise, use standard transliteration.
- **Airport Names**: Never add the word "airport" if it leads to duplication (e.g., "Letisko Paríž Charles de Gaulle" is redundant; "Paríž Charles de Gaulle" is preferred).

## Content Type Guidelines
- **UI**: Short, clear, action-focused. One idea per string.
- **Marketing**: Natural, fluid, inspirational. Focus on traveler benefits (time, money, discovery).
- **SEO**: Natural language only; no keyword stuffing.
- **Google Ads**: Concise; clear benefits; direct CTA. No "best ever" superlatives or clickbait.
- **App Store**: Benefits-focused. No guarantees.
- **Editorial**: Warm, descriptive, factual, practical.
- **Legal/Policies**: Precise, neutral, consistent. No emotional language.

## Brand, Legal, & Sustainability
### The use of "Best"
- **Avoid** claiming "best prices" or "best deals" (unsubstantiated).
- **Exception**: If a clear disclaimer explaining how "best" is measured exists, it may be used.
- **Alternative**: Use "epic prices" or "amazing deals."

### The use of "Discount"
- **Never** claim "Skyscanner offers" a discount.
- **Phrasing**: Use "You are offered a discount" or similar neutral phrasing.
- **Accuracy**: Do not use "discount" for price fluctuations/deals; reserve for explicit partner discounts.

### Sustainability Content
- **Comparatives Only**: Use "X is more sustainable than Y." Never imply any flight is fully "sustainable."
- **No Absolutes/Greenwashing**: Never use "the most ecological," "green," "eco," "clean," "planet-friendly," or "guilt-free."
- **Data-Driven**: Base claims on measurable, verified data. Avoid vague nature-themed imagery.

## Common Pitfalls & Best Practices

| Category | Do | Don't |
| :--- | :--- | :--- |
| **Tone** | "Porovnajte si lety..." | "Varovanie: Rýchlosť prehliadača..." (Overly dramatic) |
| **Formatting** | Bold/Italics for emphasis only | ALL CAPS, excessive punctuation |
| **CTAs** | Imperative (e.g., "Rezervujte") | Infinitive (if awkward) or passive |
| **Sustainability** | "o 20 % menej uhlíka" | "Tento let je ekologický" |
| **Names** | "Paríž Charles de Gaulle" | "Letisko Paríž Charles de Gaulle" (Redundant) |
| **User Address**| "Ahoj, (meno)..." (Consistent) | Mixing informal/formal unnecessarily |
| **UI Clarity** | "Zobraziť ponuky letov" | "Zobraziť výber dostupných možností..." (Wordy) |
| **Numbers** | "o 20 %" | "o 20%" (Use space before %) |

## Glossary & Terminology Management
- **Feature Names**: Prioritize meaning over literal translation. If English is standard/known (e.g., "Drops"), keep it, unless a natural local equivalent is superior.
- **Checklist before finalizing**:
    1. Is it short and easy to scan?
    2. Is it clear?
    3. Is it natural in Slovak?
    4. Is it consistent with local travel competitors?
    5. Could it be misinterpreted? (e.g., "Packages" vs "parcels").

## Accessibility & Localization
- **Emojis**: Use sparingly. Support clarity; never replace key words. Never use in legal/serious messages.
- **Imagery**: Flag culturally inappropriate references (clothing, gestures, food, etc.).
- **Resources**: Use `slovnik.juls.savba.sk` and `jazykovaporadna.sme.sk` for verification.
