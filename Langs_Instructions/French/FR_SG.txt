# FR Style Guide for Skyscanner

## Critical Requirements
**Priority Hierarchy**: Be **Helpful** (foundation), then **Geeky** (data-driven/human), then **Playful** (tone/energy). If forced to choose, prioritize **Helpful**.

**Tone of Voice**:
- **Helpful**: Simplify complex info, lead with benefits, be honest/transparent. Use 1st person plural ("nous") for Skyscanner.
- **Geeky**: Show expertise/curiosity, explain data clearly, enthusiastic but controlled.
- **Playful**: Conversational, light energy, use travel hacks. **Avoid**: Corporate jargon, overly formal/stiff phrasing, "LOL-funny" attempts.
- **Goal**: Sound like a savvy, friendly travel pal.

## 1. Technical & Functional Rules

### 1.1 URL Requirements
- **Format**: All `lowercase`, no spaces (use hyphens `-`), no special/extended characters (use basic Latin).

### 1.2 Placeholders, Variables & Tags
- **Handling**: Never translate, delete, or change placeholders/tags (e.g., `@@city@@`).
- **Implementation**: Ensure correct order and grammar. If syntax requires, use periphrasis or reordering to ensure natural French without breaking variable usage. Note: Country variables do not include the article.

### 1.3 Inclusive Language
- **Avoid**: Stereotypes, gendered language (use bracketed forms if unavoidable), assumptions about the traveler.
- **Preference**: Use neutral, precise terms.

## 2. Punctuation, Grammar & Formatting

### 2.1 Punctuation
- **Apostrophes**: Use straight apostrophes `'`.
- **Ellipsis**: Use three dots `...` (no typographic symbol).
- **Guillemets**: Use `« »` with non-breaking spaces.
- **Exclamation Marks**: Use sparingly. Avoid multiple marks.
- **Dashes**: Avoid en/em-dashes. Use standard punctuation or new sentences.
- **Full Stops**:
    - **Do NOT use** in CTAs or titles.
    - **MUST use** in subheaders and body copy.
    - **Lists**: Use full stops if items are phrases; do not use if items are single words/concepts.
- **Non-breaking spaces**: Required before `: ; ! ? %` and within dates/names/units.

### 2.2 Formatting
- **Numbers**: Use standard French format (comma for decimals, space for thousands).
- **Time**: Use 24h format.
- **Superscripts**: Do not use.
- **Capitalization**:
    - Use accented characters on caps.
    - Titles/CTAs: Capitalize first word only, rest lowercase (except proper nouns).
    - Colon: Use lowercase letter after.
    - Months/Days: Lowercase unless in isolation.
    - Nicknames (Geo): Capitalize every word.
- **Address Policy**: Use formal **"vous"** consistently across all content.

## 3. Linguistic Guidelines

### 3.1 Terminology & Brand
- **Skyscanner**: Never translate, never add articles.
- **"Best"**: Avoid superlatives unless a disclaimer explains the measurement method. If unverified, use "prix avantageux" or "super offres".
- **Discounts**: **Never** claim "we offer a discount." External partners provide discounts. Use neutral phrasing: "You are offered a discount."
- **Anglicisms**: Avoid unless widely accepted (e.g., brand names, tech terms). Ensure consistency.
- **Abbreviations**: Use only if widely understood. Write out full term first + abbreviation in brackets if unclear.

### 3.2 Sustainability (Greenwashing Avoidance)
- **Avoid**: Vague buzzwords ('green', 'eco', 'planet-friendly', 'guilt-free').
- **Requirement**: Use factual, data-based comparatives.
    - *Example*: Use "20% less carbon" instead of "a greener flight".
- **Focus**: Transparent, measurable impact.

### 3.3 Content-Specific Directives
- **UI**: Short, clear, action-focused. 1 idea per string.
- **Marketing**: Natural, fluid, inspirational.
- **SEO**: Natural language, no keyword stuffing.
- **Ads**: Concise, clear benefits, direct CTA. Avoid clickbait.
- **Legal**: Neutral, precise, no emotional/idiomatic language.
- **Surveys**: Use neutral, unbiased phrasing (bipolar/unipolar scales).

## 4. Representative Examples (Do/Don't)

| Feature | Do | Don't |
| :--- | :--- | :--- |
| **Tone** | Comparez des vols sans ouvrir trop d’onglets. | Vous vous perdez entre trop d'onglets ? Respirez. |
| **Punctuation** | New York vous attend : réservez. | New York vous attend — réservez. |
| **CTAs** | Réserver maintenant | Réserver Maintenant |
| **Formatting** | 20 % | 20% (with space) |
| **Geo** | Aéroport Paris Charles de Gaulle | Aéroport Paris Charles de Gaulle International |
| **"Best"** | Prix avantageux | Meilleur prix au monde (unverified) |
| **Sustainability** | Émet 20 % de carbone en moins. | C’est le choix le plus vert. |
| **Structure** | Cliquez sur « Revenir aux résultats ». | Cliquez sur REVENIR AUX RÉSULTATS. |
| **Address** | Vous pouvez chercher un vol | Tu peux chercher un vol |

## 5. Resources for Transliteration/Names
- Official French government recommendations (states, capitals).
- Wikipedia (Typographical conventions, geographical lists, UNESCO, etc.).
- Use official local names for airports/cities; avoid outdated forms.
- **Airport Names**: Do not add "airport" if redundant; "international" goes at the end.
